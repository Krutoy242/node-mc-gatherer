# mc-gatherer

Gather recipes from different Minecraft sources to use in https://github.com/Krutoy242/CraftTreeVisualizer

## Requirments

Here's what you need to set up in your modpack before you run the tool:

- Mod *JEIExporter*
  > Main engine, but could be omited. In this case, not names and mostly no recipes would be generated
- Mod *CraftTweaker* and special scripts
  > This scripts attached to tool. They would generate additional data, like mining levels
- Mod *Just Enough Resources*
  > Generates most of items that could be found in world
- Mod *Just Enough Calculation* 
  > This mod used to create custom recipes if needed
- Mod *TellMe* 
  > Could easily output many Minecraft information, like oredicts or biome mob spawns
